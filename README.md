# Yiddish Klal Keyboard for iOS

A QWERTY-based keyboard for typing in Standard Yiddish (YIVO orthography) on iOS.

## Features

- Full YIVO-standard character set with proper diacritics
- QWERTY layout following English keyboard positions
- Shift or long-press for alternate characters
- Customizable backspace behavior
- Numbers, symbols, and punctuation

## Installation

Download from the [App Store](https://apps.apple.com/) (link will be added after approval)

Then enable the keyboard in Settings → General → Keyboard → Keyboards → Add New Keyboard → Yiddish Klal

## Privacy Policy

This app does not collect, transmit, or store any data you type. All processing happens locally on your device.

This app does not use any third-party analytics, tracking, or advertising services.
